# Images Directory

Add your images to this folder with the following names for the modern portfolio design:

## Required Images:

1. **profile.jpg** - Your professional profile photo (recommended: 400x400px, square)
   - Displayed in a circular frame with animated ring in the hero section
   - If not provided, a beautiful gradient placeholder with your initials will show

2. **about-work.jpg** - Professional workspace or tech environment image (recommended: 800x600px)
   - Displayed in the About section with overlay effects
   - Shows your work environment and professional setting

3. **skills-tech.jpg** - Technology stack or workspace image (recommended: 1200x600px)
   - Displayed prominently in the Technical Skills section
   - Showcases your technical environment

4. **experience.jpg** - Work experience or office image (recommended: 400x500px, portrait)
   - Displayed alongside your experience card
   - Represents your professional journey

## Project Images:

5. **project-rag.jpg** - RAG/AI chatbot or AI system image (recommended: 800x600px)
   - Featured project image for RAG systems
   - Should represent AI/ML technology

6. **project-fraud.jpg** - Fraud detection or security image (recommended: 600x400px)
   - For the Fraud Detection System project card

7. **project-cnn.jpg** - Image classification or deep learning visualization (recommended: 600x400px)
   - For the CNN Image Classification project card

8. **project-sentiment.jpg** - Sentiment analysis or NLP visualization (recommended: 600x400px)
   - For the Sentiment Analysis Platform project card

9. **project-maintenance.jpg** - Predictive maintenance or industrial tech image (recommended: 600x400px)
   - For the Predictive Maintenance project card

10. **project-recommendation.jpg** - Recommendation system or e-commerce image (recommended: 600x400px)
    - For the Recommendation System project card

## Image Tips:

- **Format**: Use JPG or PNG format
- **Quality**: High resolution but optimized for web (aim for 200-500KB per image)
- **Aspect Ratios**: Follow recommended dimensions for best display
- **Fallback**: Images will gracefully hide if missing - the design works perfectly without them
- **Performance**: All images use lazy loading for optimal performance
- **Style**: Modern, professional images work best with the trending design aesthetic
